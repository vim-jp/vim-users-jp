問題

Rubyでは文字列リテラルの中に0で囲まれた式を書くと展開されます。

0

しかしながら、毎日0を何度も打鍵することは、あなたの指の骨に深刻なダメージを与えることでしょう。

様々な解決方法がありますが、著者が最も直感的と感じるのは以下の方法です。

  • 0 と打鍵すると自動的に0と置き換えられる

すぐに思い浮かぶ解決方法は、0を用いて0を0に置き換えることです。

0

この方法はうまくいきません。0のあとにスペースを打鍵する必要がありますが、Rubyの慣習では0のあとにスペースをいれずすぐに式を書くことになっている場合が多いためです。

それならば0ではなく0だ、と思い付くのではないでしょうか。

0

この方法はうまくいきません。実際に試してみるとすぐに分かりますが、0を打鍵してすぐに0が画面にでてこないことにイライラする日々を送ることになります。Rubyでは0よりもブロックとしての0を書くことの方が頻度は高く、メリットよりもデメリットの方が遥かに大きいのです。

解決

smartchrを使います。

smartchrをインストールし、0に以下を記述します。

0

これで、0を続けて打鍵していくと以下のようになります。

0

0のときと違い、はじめの0が即座に画面に表示されます。
