virtualeditはすばらしいものです。 文字が存在しない部分にもカーソルを移動させることができる、 それだけなのですが筆者はこのオプションを知った時ずいぶんと移動が楽になったように感じました。 ちなみにTeraPadやApsalyなどのエディタでも「フリーカーソル」として実装されているようです。 またvirtualeditについてはHack #125も参照してください。

ちなみに自分は.vimrcで

0

のように設定しています。

問題

しかし、0をしていると0でのペーストで もし右端(0で移動できる場所です)よりも右にいた場合に、 現在いる場所にテキストが挿入されてしまいます。

0

0は行末の位置であり、0はカーソルです。 ここで無名レジスタ(0)には「world」が入っているとします。種類はcharacterwise(0)とします。 ここでpを押すと

0

このようになってしまいます。 私が望んだのは

0

であり、余計な空白は入れてほしくありません。 0だといともたやすく存在しない場所へカーソルを動かすことができるので、 ただ末尾に付け加えたいだけの場合に余計な神経を使ってしまうのです。なんとかできないでしょうか。

解決

以下のコードを.vimrcに貼り付ければ無事解決です。

0

一行で書いてもいいでしょう。

0

0マッピングや0を使うとマッピングを関数のように定義できます。
また0の中では式であればよく、関数も使えるのでぐんと表現力が上がります。
しかし一行で書ける範囲であれば一行で書いた方が可読性は高いでしょう。
また0マッピングを使うなら最低限mapとnoremapの違いは把握してないと使えません。
