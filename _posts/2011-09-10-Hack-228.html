問題

vim を長い間使用していると、お気に入りの定番カラースキームが３、４種類は決まってくるものです。

お気に入りのカラースキームを探す場合は、

  1. Color Sampler Pack をインストールする。
  2. 0 を駆使して片っ端から気に入るものを探す。

という流れで探すのが便利で手軽だと思います。

0 によるカラースキームの変更は一つ一つ 0 コマンドでカラースキームを入力する手間に比べればとても手軽です。
しかし、一度お気に入りが決まってしまうと、 もっと手早くカラースキームを切り替えたいと感じる方もいるのではないでしょうか？

問題提起が少し強引な気もしますが、筆者はお気に入りの３、４種類のカラースキームをワンキーで手早く切り替えたいと感じていました。

対処

下記のコードを 0 に貼り付けます。

0

解説

お気に入りのカラースキーム群をローラー( 0 )に見立てて、ローラーを回す( 0 )、戻す( 0 )ことで、カラースキームを切り替えます。
0 を 0 、 0 を 0 (Shiftキーを押しながら 0 )にそれぞれ割り当てています。

使い方

お気に入りのカラースキームを、 0 に設定して使用します。

tabpagecolorscheme を使用している場合、 0 内のコメントを参考に、
0 の行を 0 に変更して使用して下さい。
